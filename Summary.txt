# My Markdown Document

## Introduction
This is a simple example of writing Markdown in a GitHub text file. Markdown is a lightweight markup language that allows you to format text using simple syntax.

## Features
- **Headers**: You can create headers using the `#` symbol. The number of `#` symbols determines the level of the header.
- **Lists**: You can create ordered or unordered lists using `-` or `1.` respectively.
- **Links**: You can create links like [this link](https://www.example.com) using `[link text](URL)`.
- **Images**: You can embed images like ![this](https://via.placeholder.com/150) using `![alt text](URL)`.
- **Emphasis**: You can add emphasis to text using `*italic*` or `**bold**`.
- **Code Blocks**: You can display code blocks using triple backticks:
